## ğŸš€ Installation Instructions

### For Chrome/Edge/Brave:

1. **Create the folder structure** as shown above
2. **Copy all files** into the `SafeGuard` folder
3. **Create icons** (see Icons section below)
4. Open your browser and go to:
   - **Chrome:** `chrome://extensions/`
   - **Edge:** `edge://extensions/`
   - **Brave:** `brave://extensions/`
5. **Enable "Developer mode"** (toggle in top right)
6. Click **"Load unpacked"**
7. Select the `SafeGuard` folder
8. Done! The extension is now installed âœ…

### For Firefox:

1. Create the folder structure and copy all files
2. Go to `about:debugging#/runtime/this-firefox`
3. Click **"Load Temporary Add-on"**
4. Select the `manifest.json` file from the SafeGuard folder
5. Done! âœ…

## ğŸ¨ Creating Icons

You need 3 icon sizes. You can:

### Option 1: Use a design tool
- Create a shield icon in **Figma**, **Canva**, or **Photoshop**
- Export as PNG in 3 sizes: 16x16, 48x48, 128x128
- Save them in the `icons/` folder

### Option 2: Use an online tool
- Go to [favicon.io](https://favicon.io/favicon-generator/) or similar
- Generate icons with "SG" or a shield symbol
- Download and rename to: `icon16.png`, `icon48.png`, `icon128.png`

### Option 3: Simple placeholder
Use any square PNG image and resize it to the 3 required sizes.
